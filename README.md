# MamkatStore – African Fashion & Textiles eCommerce MIS

A complete, database‑driven eCommerce Management Information System built with **PHP**, **MySQL**, and vanilla **JavaScript**.  
RedStore specialises in handbags, textiles, laces, getzner, abayas, children's wear, and African traditional attires.

## Features

- **100+ products** across 9 curated categories
- **User authentication** – manual (bcrypt) + Google OAuth 2.0
- **Shopping cart** – persistent session‑based cart
- **Checkout** with multiple payment gateways:
  - Cash on Delivery
  - Stripe (Card)
  - Monnify (Moniepoint)
  - Flutterwave
- **Order tracking** with real‑time status updates
- **Email notifications** via PHPMailer
- **Customer testimonials** (moderated by admin)
- **Admin panel** with dashboard, CRUD for products/users/orders, analytics, low‑stock alerts
- **Responsive design** – works on desktop and mobile
- **Rate limiting**, CSRF protection, input sanitisation

## Tech Stack

- **Backend:** PHP 8 (no framework), MySQL
- **Frontend:** HTML5, CSS3, Vanilla JS, Chart.js, Swiper.js
- **Libraries:** Google API Client, Stripe PHP, PHPMailer, Monnify/Flutterwave APIs
- **Tools:** Composer, Git, XAMPP/LAMP

## Installation

### Prerequisites

- XAMPP (Apache + MySQL + PHP) or LAMP stack
- PHP 7.4+ with `curl`, `gd`, `mysqli` extensions
- Composer
- Git
